# Custom Gatsby blog starter

## Setup

```
$ gatsby new PROJECT_NAME https://github.com/jumpalottahigh/gatsby-starter-blog-jumpalottahigh
```

## Development

```
$ npm run dev
```

## Production build

```
$ npm run build
```

## Features

- SEO optimizations

  - open graph
  - google structured data
  - twitter cards

- UI

  - article list + image
  - search
  - filter by category
  - mobile friendly FAB + Drawer menu
  - reader feedback component
  - reading progress line component
  - related articles component

- sitemap
- rss feed
- social media links
- google analytics
- google tag manager
- prismjs
